# UI-песочница

Локальная лаборатория для экспериментов с дизайном. **Не попадает в git** — папка `src/app/[locale]/sandbox/` добавлена в `.gitignore`.

## Быстрый старт

```bash
npm run sandbox:init
npm run dev
```

Откройте: [http://localhost:3000/ru/sandbox](http://localhost:3000/ru/sandbox)

Раздел **Шрифты** — IBM Plex Sans, Iosevka Charon (через `@fontsource`), Libre Franklin и смешанные пары. Переключатель меняет `--font-sans` / `--font-heading` для всей песочницы.

## Команды

| Команда | Действие |
|---------|----------|
| `npm run sandbox:init` | Создать песочницу из шаблона (если ещё нет) |
| `npm run sandbox:init -- --force` | Сбросить песочницу к шаблону (удалит ваши правки!) |

## Как устроено

- **Шаблон** (в git): `sandbox.template/app/[locale]/sandbox/`
- **Рабочая копия** (не в git): `src/app/[locale]/sandbox/`

Меняйте что угодно в рабочей копии — кнопки, карточки, новые секции, шрифты. Когда результат понравится, переносите идеи в основные компоненты вручную.

## Безопасность

В production маршрут `/sandbox` возвращает 404 — даже если папку случайно закоммитят.

## Новая машина / коллега

```bash
npm run sandbox:init
```

Шаблон всегда в репозитории, рабочая копия создаётся локально.
